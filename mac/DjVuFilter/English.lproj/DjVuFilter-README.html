DjVuFilter.service
v1.1

Introduction

DjVu is a relatively new image and document format developed at AT&T and acquired by LizardTech (http://www.lizardtech.com/). This version of DjVuFilter.service is a departure from the old version as it is now linked against libdjvulibre (http://djvu.sourceforge.net/).

Installation

To install DjVuFilter.service, just drag the icon from the to one of the following locations:

  • /Library/Services
  • ~/Library/Services
You will need to log out an log back in before being able to use the service. (Please, if someone else knows the command that will refresh Services without logging out and back in, forward that to the contact info below)

Known limitations

DjVuFilter.service is a very limited decoder of DjVu and IW4 files. It works as a background filter for images in any Cocoa application that accepts NSTIFFPboardType (OmniGraffle, Create, PixelEnhance,...). For multi-page DjVu files, DjVuFilter will only convert the first page. This limitation is also of note if you are using some of the web sites that vend DjVu files as the index.djvu will likely not render anything as the additional files referenced will not be available on your local file system (see http://www.djvuzone.org/ for more details about the multi-page DjVu document format).

Source code

The source requires djvulibre-3.5.13 and libjpeg.a. Copy the src/DjVuFilter directory into djvulibre-3.5.13/mac and then build with Xcode (v1.2 and gcc 3.3).
